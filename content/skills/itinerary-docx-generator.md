## Description

Generate professional Chinese travel agency itinerary documents (行程单) from simplified input. Fills a standardized DOCX template with trip details including cities, dates, hotels, transportation, and daily activities while preserving the template's professional formatting. Essential for visa applications that require detailed travel itineraries, especially for Chinese visa submissions where properly formatted documents are expected.

## Description (中文)

从简化的输入信息生成专业的中国旅行社行程单文档。使用标准化 DOCX 模板填充城市、日期、酒店、交通和每日活动详情，同时保持模板的专业格式。特别适用于需要详细行程单的签证申请，尤其是中国签证提交时需要格式规范的行程文件。
