## Description

Professional bidirectional Chinese-English translation with detailed grammar analysis and language learning insights. Beyond simple word-for-word translation, this skill provides structured breakdowns including grammatical nuances, tense corrections, idiomatic expressions, and common "Chinglish" error patterns. Ideal for foreigners learning Chinese, developers localizing content, and travelers needing accurate translations with linguistic context.
